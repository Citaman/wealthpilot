**Intent**: AI-first personal finance UI that feels calm, premium, and kinetic. The shell shifts from a left sidebar to a bottom command dock. The dock anchors navigation, quick tools, notifications, and intelligent search in one place.

**Reference cues**:
- `images_ref/image.png` for bottom dock and calm surface layering.

---

## Design Pillars

1) **Calm intelligence**
   - Soft depth, minimal borders, no heavy shadows.
   - Blurred overlays only for modals and the command dock.
2) **Command-first**
   - Search is the primary interface element, not a utility.
   - Navigation and quick tools cluster around the command bar.
3) **Data clarity**
   - Numeric hierarchy is strong and legible at a glance.
   - Tabular numerals everywhere money appears.
4) **Motion with purpose**
   - Small, confident transitions that clarify hierarchy.
   - Reduced-motion support is mandatory.

---

## Layout & Shell

### Global layout
- No permanent sidebar.
- Content sits in a soft canvas with a faint gradient wash.
- A floating top "page header" inside the content area shows title and key actions.

### Bottom Command Dock
- Fixed, centered, floating dock (like macOS) with three zones:
  - **Left**: Core nav (Dashboard, Transactions, Analytics, Budgets, Goals).
  - **Center**: AI Command input (primary), with modes: Search, Action, Ask.
  - **Right**: Notifications, Quick Tools, Settings, Profile.
- Dock styling:
  - Frosted glass background with a thin keyline.
  - Subtle glow around the active nav icon.
  - Dock scales slightly on hover (1.02) and compresses on click (0.98).

### AI-first Command Bar
- Placeholder: "Ask WealthPilot or search..."
- Chips inside the bar: "Search", "Transfer", "Import", "Budget".
- Supports keyboard shortcuts: Cmd/Ctrl+K opens; Enter executes; Esc closes.

## Component Contracts

### Cards
- Padding: 16/20
- Radius: 16
- Border: 1px keyline (subtle)
- Shadow: none by default, only on hover for primary cards

### Buttons
- Primary: filled with Aurora Teal
- Secondary: glass + keyline
- Ghost: text only, hover highlight

### Inputs
- Height: 40/44
- Focus ring: 2px Aurora Teal, no heavy shadow

### Tables
- Zebra optional (10% tint on alternate rows)
- Row height: 44 (compact 36)
- Sorting: subtle chevron, active column tint

### Badges
- Filled soft chips for status
- Outlined chips for tags

## Motion & Animation

- Page load: 180ms fade + 8px rise
- List rows: staggered 40ms between rows
- Dock hover: 120ms ease-out scale
- Dialogs: 140ms fade + 6px rise
- Reduced motion: no translate, only opacity

## Mascot / Brand Mark

**Concept**: "Pilot Orb"
- Minimal circular mark with a compass notch.
- Used in the dock, empty states, and onboarding.
- Optional micro-variant: an "eye" dot to imply guidance.

---

## Do / Don't

Do:
- Keep strong numeric hierarchy.
- Use glass only for overlays and dock.
- Keep surfaces calm and neutral.

Don't:
- Overuse glow or blur.
- Put gradients behind dense tables.
- Use neon or overly saturated primaries.
